
A multi-purpose tool for various tasks like cryptography, steganography, Wi-Fi hacking, password cracking, and more.

- **Cryptography**: AES encryption and decryption.
- **Steganography**: Hide and extract data in images.
- **Wi-Fi Hacking**: Capture handshakes and crack passwords.
- **Nmap Scanning**: Host discovery, OS detection, port scanning, and more.
- **Whois Lookup**: Perform Whois queries.
- **Password Cracking**: Crack passwords using John the Ripper.
